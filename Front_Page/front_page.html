Phish Detective

How to Play:
  1. Press the Play button to get started.
  2. You will be prompted to enter your name and press submit, but you can skip if you would not like your name to be used in questions.
    You can also try out the Password Assessment Tool which is optional.
  3. You will then be given a set of scenarios (emails, texts, direct messages) to analyze along with context for each one.
    Determine whether each scenario represents a phishing attempt or if it is legitimate/real.
  4. After answering each question, press the Finish button to see your score.
  5. Press Play Again if you would like to play another round.
